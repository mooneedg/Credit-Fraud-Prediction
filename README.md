# Credit Fraud Prediction

## Overview
This project focuses on identifying fraudulent credit card transactions using machine learning. Due to the highly imbalanced nature of fraud datasets, this notebook emphasizes predictive accuracy and effective evaluation metrics over standard accuracy scores.

## Methodology
* **Data Processing:** Imported and analyzed a dataset of 284,807 transactions.
* **Modeling:** Trained a `RandomForestClassifier` to identify fraudulent activities.
* **Evaluation:** Evaluated the model using advanced metrics including ROC-AUC scores, Precision/Recall curves, and Confusion Matrices to properly account for class imbalances.

## Technologies & Libraries
* **Environment:** Google Colab
* **Libraries:** Pandas, NumPy, Seaborn, Matplotlib, Scikit-Learn
